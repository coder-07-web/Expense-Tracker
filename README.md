# 💰 Expense Tracker (MERN Stack)

A full-stack Expense Tracker web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It helps users manage their income and expenses efficiently with visual insights and secure authentication.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 👤 Profile Image Upload
- ➕ Add / Delete Income & Expenses
- 📊 Interactive Charts (Bar, Line, Pie)
- 📅 Track last 30 / 60 days data
- 📥 Download data as Excel
- 📱 Fully Responsive UI
- 💾 Secure data storage with MongoDB

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Recharts
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Other Tools
- JWT Authentication
- Multer (File Upload)
- XLSX (Excel Export)
